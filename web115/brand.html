Brand Name: Bümulaz

Bümulaz is one of the many usernames I have held over the years. I feel that it gives the page a more interesting sound to it rather than just using my own name.

The "company" name that I chose for the website is BümuEnterprises so that I could add a trademarked name for the page to be owned by.
